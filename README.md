

# 🔒 WiFi Security Auditor

A Python-based WiFi Security Auditor developed for educational purposes. This project scans nearby WiFi networks and displays useful information for basic wireless security analysis.

> **Note:** This project is intended only for learning and authorized security auditing.

---

## Features

- Scan available WiFi networks
- Display SSID (Network Name)
- Display Signal Strength
- Display Security Type
- Fast and lightweight
- Command-line interface
- Built using Python

## Installation

Clone the repository:

```bash
git clone https://github.com/YourUsername/WiFi-Security-Auditor.git
```

Go to the project folder:

```bash
cd WiFi-Security-Auditor
```

Run the program:

```bash
python main.py
```

---

## Requirements

- Python 3.10 or later
- Windows Operating System

No external Python libraries are required.
